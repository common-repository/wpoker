== Description ==

Convert [As] [10d] to graphics

== Installation ==

= Minimal Requirements =
* PHP: 5.x.x
* WordPress: 3.0

= Recommended Requirements =
* PHP: 5.2.x or newer
* WordPress: 3.0.1 or newer

= Installation =


1. Unpack the downloaded zipped file
2. Upload the "allin-poker" folder to your /wp-content/plugins directory
3. Activate it

== Use ==

This plugin will convert shortcode as [As] in your post or comments to a graphic card.
You can find examples here : http://www.wpoker.me/wpoker/preview/
